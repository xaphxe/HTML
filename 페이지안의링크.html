HTML이란?

        HTML(HyperText Markup Language)은 웹 페이지의 구조와 내용을 정의하는 기본 마크업 언어로, 
        태그를 사용하여 문단, 제목, 이미지, 링크 등의 요소를 구조화합니다. 웹 브라우저가 
        콘텐츠를 이해하고 표시하게 하는 뼈대 역할을 하며, 프로그래밍 언어가 아닌 마크업 언어입니다.

CSS란?

        CSS (Cascading Style Sheets)는 HTML, XML 등으로 작성된 웹 문서의 글꼴, 배경색, 크기, 
        위치 등 겉모습(스타일)을 기술하는 스타일 시트 언어입니다. 웹 콘텐츠의 구조(HTML)와 디자인(CSS)을 
        분리하여 관리하게 해주며, 브라우저, 기기별로 화면이 다르게 표시되도록 렌더링을 제어합니다

JS란?

        JavaScript(JS)는 웹페이지를 동적이고 대화형으로 만드는 경량, 인터프리터 프로그래밍 언어입니다. 
        HTML/CSS와 함께 웹 프론트엔드의 핵심 기술로, 애니메이션, 데이터 업데이트 등 생동감을 부여하며, 
        Node.js를 통해 서버 환경에서도 사용되는 객체 기반 스크립트 언어입니다.
